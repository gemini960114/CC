# 專案實戰：使用 Guizang PPT Skill 製作電子雜誌風網頁簡報

## 1. 課程簡介與學習目標

在現代的工作與分享場景中，一份具備高度設計感、能迅速抓住眼球的簡報至關重要。本堂課將帶領大家學習如何使用 **Guizang PPT Skill**，透過 AI 代理（如 Claude Code）快速生成**單一檔案 HTML 的橫向翻頁簡報**。

這個 Skill 的視覺基調是「**電子雜誌 × 電子墨水**」，排版風格類似知名雜誌 *Monocle*。

### 學習目標：
*   了解如何安裝與觸發外部 Skill。
*   掌握 Guizang PPT 的核心設計原則與適用場景。
*   學會透過對話引導 AI 完成結構化的 6 步工作流，產出精美的網頁簡報。

### 適用與不適用場景
*   ✅ **非常適合**：線下分享、私享會、AI 產品發布、Demo Day、具強烈個人風格的演講。
*   ❌ **不適合**：大量表格數據的財報、需要高資訊密度的培訓課件、需要多人協作編輯的場景。

## 2. 準備與安裝

要開始使用，我們需要先讓 Claude Code 學會這個技能。

### 安裝方式
請在你的命令列或 Claude Code 環境中執行以下任一方式：

**方式一：透過 npx 快速安裝（推薦）**
```bash
npx skills add https://github.com/op7418/guizang-ppt-skill --skill guizang-ppt-skill
```

**方式二：對話式安裝**
直接將以下這段話貼給具備 Shell 權限的 AI Agent（如 Claude Code / Cursor）：
> 幫我安裝 `guizang-ppt-skill` 這個 Claude Code skill。請按下面步驟做：
> 1. 確保 `~/.claude/skills/` 目錄存在 (不存在就創建)
> 2. 執行 `git clone https://github.com/op7418/guizang-ppt-skill.git ~/.claude/skills/guizang-ppt-skill`
> 3. 驗證：`ls ~/.claude/skills/guizang-ppt-skill/` 應該看到 `SKILL.md`、`assets/`、`references/` 三項
> 4. 告訴我安裝好了，之後我說「做一份雜誌風 PPT」之類的話就會觸發這個 skill

### 觸發方式
安裝完成後，只要在對話中輸入以下關鍵字，即可喚醒該技能：
*   「幫我做一份雜誌風 PPT」
*   「生成一個 horizontal swipe deck」
*   「editorial magazine style presentation」

## 3. 核心設計原則

在開始動手前，請先理解這個 Skill 的美學堅持。AI 在生成時也會遵循這些原則：
1.  **克制優於炫技**：WebGL 動態流體背景僅在封面（Hero 頁）呈現，正文頁保持乾淨。
2.  **結構優於裝飾**：資訊層級依賴「字號對比」與「網格留白」，不使用陰影或浮動卡片。
3.  **圖片是第一公民**：圖片裁切以底部為主，保留頂部與左右的完整性。
4.  **節奏靠 Hero 頁**：Hero 頁面與一般內容頁面交替出現，確保觀眾視覺不疲勞。
5.  **嚴格的主題色控制**：內建 5 套配色，**不允許自定義 hex 色值**，保護美學不被破壞。

## 4. 實戰演練：從零到一製作簡報

Guizang PPT Skill 內建了結構化的 **6 步工作流**，一旦觸發，Claude 會逐步引導你完成。

### Step 1: 需求澄清 (6問清單)
當你請 AI 製作簡報時，它首先會向你確認以下資訊：
1.  受眾是誰？
2.  演講時長？
3.  要放哪些文字素材？
4.  需要哪些圖片？
5.  選擇哪一套主題色？
6.  有哪些硬性約束？

**💬 實戰提示：** 你可以一次性把準備好的講稿與上述問題的答案丟給 AI，例如：「我要做一份 5 分鐘的產品介紹，受眾是投資人，請用『靛藍瓷』主題，素材在...」

### Step 2: 建立專案與選擇主題色
AI 會自動從套件中複製 `template.html` 到你的專案目錄，並根據你的選擇套用主題。
目前提供 5 套預設：
*   🖋 **墨水經典** (通用預設)
*   🌊 **靛藍瓷** (科技/AI)
*   🌿 **森林墨** (自然/文化)
*   🍂 **牛皮紙** (懷舊/人文)
*   🌙 **沙丘** (藝術/設計)

### Step 3: 填充內容與版面配置
這是一份 HTML 簡報，AI 會根據你的內容，從 **10 種預設版面 (Layouts)** 中挑選最合適的結構進行填充：
*   開場封面 / 章節幕封 / 數據大字報 / 左文右圖 / 圖片網格 / Pipeline / 懸念問題 / 大引用 / Before-After 對比 / 圖文混排

### Step 4: 自我檢查 (Checklist)
AI 在生成完程式碼後，會自動對照 `checklist.md` 進行 P0 級別的致命錯誤檢查（例如是否擅自新增了未定義的 CSS 類別）。

### Step 5 & 6: 預覽與迭代微調
產生出來的會是單一的 `index.html` 檔案。
*   **預覽**：直接在瀏覽器雙擊打開，不需啟動伺服器。支援鍵盤左右鍵、滑鼠滾輪或觸控滑動翻頁。
*   **迭代**：如果覺得字體太大或間距不對，可以請 AI 透過 Inline Style (內聯樣式) 進行微調。

## 5. 進階技巧與注意事項

*   **本機圖片處理**：簡報若要包含圖片，建議將圖片與 HTML 檔放在同一資料夾，並請 AI 使用相對路徑引入。你也可以使用網路圖片連結。
*   **不要要求 AI 重寫底層 CSS**：該 Skill 的 CSS 已經過高度最佳化（包含字體排印、動畫與響應式）。請專注於「內容與排版」，不要要求 AI 修改底層邏輯，以免版面崩壞。
*   **單文件優勢**：做完的 HTML 檔案可以直接透過 E-mail 寄給他人，對方只要有瀏覽器就能觀看，體驗極佳。

## 6. 課後作業

請使用 `guizang-ppt-skill` 為自己製作一份 **「自我介紹」** 的專屬簡報。
**要求：**
1.  總頁數包含封面約 5 頁。
2.  選擇「牛皮紙」或「沙丘」主題。
3.  必須包含至少一種「大引用」版面與「左文右圖」版面。
4.  產生完成後，用瀏覽器打開親自體驗翻頁的流暢感！
